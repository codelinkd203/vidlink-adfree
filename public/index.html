API Documentation

Embed Movies

TmdbId is required from The Movie Database API.

/movie/{tmdbId}

Code Example:

<iframe src="/movie/672" frameborder="0" allowfullscreen ></iframe>

Embed Shows

TmdbId from The Movie Database API. Season and episode required.

/tv/{tmdbId}/{season}/{episode}

Code Example:

<iframe src="/tv/58981/1/1" frameborder="0" allowfullscreen ></iframe>

Embed Anime

Anime embeds use an AniList ID, episode number, and stream type (sub or dub).

<CURRENT_URL>/anime/{anilistId}/{episode}/{sub_or_dub}

Code Example:

<iframe src="<CURRENT_URL>/anime/21/1/sub" frameborder="0" allowfullscreen ></iframe>

Customization Parameters

Append parameters to the URL. First parameter starts with ?, additional ones separated by &. Use hex color codes without the #.

primaryColor

Sets the primary color of the player, including sliders and autoplay controls.

primaryColor=B20710
secondaryColor

Defines the color of the progress bar behind the sliders.

secondaryColor=170000
icons

Changes the icon design. Can be vid or default.

icons=vid
iconColor

Changes the color of the icons within the player.

iconColor=B20710
title

Controls whether the media title is displayed.

title=false
poster

Determines if the poster image is shown.

poster=true
autoplay

Controls whether media starts playing automatically.

autoplay=false
nextbutton

Shows the next episode button at 90% watch progress. Off by default.

nextbutton=true
player New

Changes player to JWPlayer or the default player.

player=jw
startAt New

Starts video at specified time in seconds.

startAt=60
sub_file New

Adds external subtitles. Must be a direct link to a VTT file.

sub_file=https://…/subs.vtt
sub_label New

Label for the external subtitle track. Defaults to 'External Subtitle'.

sub_label=English
fallback_url New

Redirect URL when the stream fails to load.

fallback_url=https://…/fallback

Watch Progress

Continue Watching Feature

Track your users' watch progress across movies and TV shows. This enables a "Continue Watching" feature on your website.

Add Event Listener

Add this script wherever your iframe lives. In React, use a useEffect.

window.addEventListener('message', (event) => { if (event.origin !== window.location.origin) return; if (event.data?.type === 'MEDIA_DATA') { const mediaData = event.data.data; localStorage.setItem('vidLinkProgress', JSON.stringify(mediaData)); } });

Stored Data Structure

Data in localStorage contains:

  • Movie/Show details (title, poster, etc.)
  • Watch progress (watched, duration)
  • Last watched episode for TV shows
  • Episode-specific progress
{ "672": { "id": 672, "type": "movie", "title": "Furiosa: A Mad Max Saga", "progress": { "watched": 8726.9, "duration": 8891.763 } } }
